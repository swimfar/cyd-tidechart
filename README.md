# cyd-tidechart
A tide chart for the Cheap Yellow Display (CYD).  Most of the original code was pulled directly from [kronsby/esp-tidechart](https://github.com/kronsby/esp-tidechart) with display and touchscreen setup info from [randomnerdtutorials.com](https://randomnerdtutorials.com/cheap-yellow-display-esp32-2432s028r/)
